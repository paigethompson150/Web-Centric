  • JavaScript

    What is it: JavaScript is a high-level, dynamic programming language that is widely used in web development, along with the web's other core technologies: HTML and CSS.

    Purpose: Allows developers to create interactive effects within web browsers, manipulate DOM (Document Object Model) and implement complex features like interactive maps and animated graphics.

    Use Case: Almost all modern websites use JavaScript to add functionality. Can run in the browser and server-side environments like Node.js

    In this project: Each game (2048, Flappy Bird, RockBand) uses JS to become responsive!

  • jQuery

    What is it: jQuery is a fast, small and feature-rich JavaScript library. Event Handling, animation and HTML document traversal are much simpler with an easy-to-use API that works across a variety of browsers.

    Purpose: Simplify client-side scripting of HTML. By abstracting complex JavaScript tasks into methods, it can be called with less code and fewer complexity.

    Use Case: Used for tasks such as selecting DOM elements, creating animations, handling events, and developing Ajax applications.

    jQuery usage has declined with the rise of modern frameworks like React, Angular, and Vue.js

    In this project: A jQuery plug-in was used for validating the contact information form and AJAX sends it as an email. jQuery was also used for the music allowing it to scroll and stop on hover.

    For Flappy Bird jQuery is used to set the text within the message class, and Ajax is used to get and display the high score from the server

  • CSS

    What is it: CSS is a language that lets us control how we present our documents to users! Description

    Purpose: Style and layout our web-pages, and control different displays for different devices or user capabilities.

    Use Case: Virtually all modern web-pages use CSS!

    In this project: CSS was used to create the design and behavior of this carousel! Each web page uses CSS, and we used the Z-Index property in the FlappyBird game to control the stacking order of elements! CSS was used to create the animation in GarageBand.
